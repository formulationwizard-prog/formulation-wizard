## Cutover-critical fixes (land in M2-1, on this verified surface)
**#2 ordering + #7 aggregation + #8 source suppression + #12 threshold** all live in the one row-building loop the cutover rewrites. Augmented spec (Opus-ratified):

> per ingredient → engine `perServingAmount` → **group by nutrient (displayName+basis)** → sum dietary-ingredient weight → **suppress (b)(2)-ingredients below the declarable threshold (<2% RDI) per (b)(2)(i)** (advise operator, like the blend-floor advisory) → **sort to CFR order** → render one row, sources in parens descending by weight, redundant "(as X)" dropped — each branch with a `// REGULATION:` anchor.
